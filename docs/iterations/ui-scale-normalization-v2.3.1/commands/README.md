# Команды (URL) для проверки масштаба

Использовать в локальном/туннельном окружении, добавляя параметры к любому маршруту.

## Основные
- Managed + Debug: `?scale=managed&debug=1`
- Legacy (откат): `?scale=legacy`

## Примеры
- `/dashboard?scale=managed&debug=1`
- `/dashboard/content/all?scale=managed&debug=1`
