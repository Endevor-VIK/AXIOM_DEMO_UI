# Скриншоты для baseline

Сюда складываем эталонные скриншоты до/после изменений масштаба.

Рекомендованный нейминг:
- `YYYY-MM-DD_resolution_context_state.png`
- Пример: `2025-12-22_1920x1080_dashboard-home_baseline.png`

Минимальный набор:
- 1920x1080 (fullscreen)
- 1600x900
- 1440x900
- 1280x720
- windowed 80-90% (2-3 критичных экрана)

## Условия съемки (важно для сравнения)
Системные скриншоты зафиксированы при других настройках масштаба, поэтому при сравнении учитывать разницу:
- Базовые скриншоты: Windows scale **150%**, browser zoom **100%**, global zoom **80%**.
- Текущие проверки/фиксы: Windows scale **125%**, browser zoom **100%**.
