# AXIOM WEB CORE UI — релиз v0.2.4.0

Версия сайта поднята с `v0.2.3.1` до `v0.2.4.0`.

## По каким поводам подняли версию
- Добавлен и расширен контур админки: отдельные admin-сессии, control center, URL command reference (`8e05222`, `9ab8d7c`, `935e66b`, `115007e`).
- Усилена надёжность авторизации и выхода: forced re-auth, стабилизация logout и audit history (`2c5f92d`, `0f9696c`).
- Добавлен live ops foundation: telemetry ingestion, snapshot/SSE, axchat JSONL audit, workspace tabs/multi-pane (`fbaa03e`, `287edc5`).
- Стабилизирован локальный dev-контур и tunnel fallback (`74f9817`, `2308fbc`, `44e07c4`).
- Доведён до стабильного состояния Orion login background (`40793a3`, `3333903`).

## Нумерация релиза
- Было: `v0.2.3.1`
- Стало: `v0.2.4.0`
- Логика bump: увеличен feature-компонент (четвёртый reset до `0`), так как изменения вышли за пределы hotfix-патча.
